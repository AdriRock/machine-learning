Домашняя работа №1

I) Создайте 4 вектора длины 100, испольщуя для генерации их элементов функции R:
1) с помощью функции sample заполнить вектор случайными числами из интервала [0,100] (числа в векторе могут повторяться)
2) с помощью функции rnorm с параметрами mean=0, sd=1
3) с помощью функции rexp с параметром rate=0.25
4) с помощью функции rgeom c параметром prob=0.25
(Так мы смоделировали значения некоторых признаков, распределенных по различным законам). Для наглядности для каждого вектора можете построить гистограмму (функция hist).

II) Для каждого вектора найти среднее значение, отклонение и медиану (функции mean,sd,median).

III)C помощью неравенства, указанного на лекции, определите, содержимое каких векторов является симметричной выборкой.

IV*) (На понимание) Пусть у выборки (вектора), заполненного способом 3), медиана меньше среднего (скорей всего, у вас действительно так и будет). Какую (очень простую) операцию нужно совершить с вектором, чтобы медиана стала больше среднего?
